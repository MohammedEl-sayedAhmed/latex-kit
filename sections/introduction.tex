\section{Introduction}
\label{sec:intro}

This is a starter LaTeX project that compiles locally with TeX Live and
VS Code (LaTeX Workshop). It mirrors an Overleaf-style workflow without
needing an Overleaf account.

A small example of math, referenced with \texttt{cleveref}:
\begin{equation}
  e^{i\pi} + 1 = 0
  \label{eq:euler}
\end{equation}
\Cref{eq:euler} is Euler's identity.
